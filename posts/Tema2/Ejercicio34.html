Ejercicio 34 — Uso de float y clear

Publicado el 21 de noviembre de 2025 por pjimpin1207

En este ejercicio se trabaja el uso de la propiedad float para alinear columnas y la propiedad clear para evitar que otros elementos, como el pie de página, se vean afectados por el flotado. Este tipo de maquetación fue muy utilizada antes de la llegada de Flexbox y Grid.

Enunciado

  • Crear una página con un encabezado, varias columnas flotantes y un pie de página.
  • Usar float para alinear las columnas.
  • Usar clear para que el pie de página quede debajo de las columnas.
  • Probar el comportamiento quitando y añadiendo clear: both.
  • Añadir una cuarta columna y ajustar los tamaños.
  • Experimentar cambiando float: left por float: right.

Explicación

Las columnas se alinean utilizando float: left, lo que hace que los elementos salgan del flujo normal del documento. Esto provoca que el elemento footer pueda colocarse incorrectamente si no se usa la propiedad clear.

Al aplicar clear: both al pie de página, se fuerza a que este elemento se sitúe debajo de todas las columnas flotantes, evitando solapamientos.

Código HTML

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Ejercicio 34</title>
  <link rel="stylesheet" href="Ejercicio34.css">
</head>
<body>

  <header>
    <h1>Ejercicio 34</h1>
  </header>

  <div class="columna izquierda">
    <h2>Columna Izquierda</h2>
  </div>

  <div class="columna central">
    <h2>Columna Central</h2>
  </div>

  <div class="columna derecha">
    <h2>Columna Derecha</h2>
  </div>

  <div class="columna extra">
    <h2>Columna cuatro</h2>
  </div>

  <footer>
    <p>pie de pagina</p>
  </footer>

</body>
</html>

Código CSS

body {
    background-color: rgb(187, 218, 204);
}

header {
    background-color: #1e90ff;
    color: white;
    border: 1px solid #000000;
}

.columna {
    width: 220px;
    margin: 10px;
    padding: 15px;
    border: 1px solid #000000;
}

.izquierda {
    background-color: #04f74d;
    float: left;
}

.central {
    background-color: #d3f704;
    float: left;
}

.derecha {
    background-color: #b204f7;
    float: left;
}

.extra {
    background-color: #f77904;
    float: left;
}

footer {
    background-color: #ff0000;
    color: white;
    clear: both;
    border: 1px solid #000000;
}

Respuestas

¿Qué sucede si no se usa clear?
El pie de página se coloca al lado o encima de las columnas flotantes, ya que estas salen del flujo normal del documento.

¿Qué otros elementos además de clear pueden solucionar problemas similares?
Se puede usar un clearfix, la propiedad overflow: hidden o técnicas modernas como display: flex o display: grid, que evitan este tipo de problemas de maquetación.
