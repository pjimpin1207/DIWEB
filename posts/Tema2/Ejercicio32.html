Ejercicio 32 — Position en CSS

Publicado el 21 de noviembre de 2025 por pjimpin1207

En este ejercicio se trabajan los distintos valores de la propiedad position en CSS. Se comparan los comportamientos de static, relative, absolute, fixed y sticky mediante ejemplos visuales dentro de un mismo documento.

Enunciado

  • Crear un contenedor con varios elementos internos.
  • Aplicar position: static y explicar su comportamiento por defecto.
  • Usar position: relative para desplazar un elemento.
  • Aplicar position: absolute dentro de un contenedor.
  • Usar position: fixed para un elemento fijo en pantalla.
  • Usar position: sticky para anclar un elemento al hacer scroll.

Código HTML

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Ejercicio Position</title>
    <link rel="stylesheet" href="position.css">
</head>
<body>

    <div class="contenedor">

        <div class="caja-static">
            <strong>Static</strong><br>
            Este elemento usa <b>position: static</b>.
        </div>

        <div class="caja-relative">
            <strong>Relative</strong><br>
            Caja con <b>position: relative</b>.
        </div>

        <div class="caja-absolute">
            <strong>Absolute</strong><br>
            Posicionada respecto al contenedor.
        </div>

        <div class="caja-sticky">
            <strong>Sticky</strong><br>
            Se queda pegada al hacer scroll.
        </div>

    </div>

    <div class="caja-fixed">
        <strong>Fixed</strong><br>
        Siempre visible en pantalla.
    </div>

    <div class="texto-scroll">
        <p>Mucho texto para permitir hacer scroll...</p>
        <p>BRRRRRRRRRRRRRRRRRRRRRRRRRRRRRRRRRR</p>
        <p>BRRRRRRRRRRRRRRRRRRRRRRRRRRRRRRRRRR</p>
        <p>BRRRRRRRRRRRRRRRRRRRRRRRRRRRRRRRRRR</p>
    </div>

</body>
</html>

Código CSS

/* 1. position: static */
.caja-static {
    position: static;
    background-color: rgb(0, 255, 42);
}

/* 2. position: relative */
.caja-relative {
    position: relative;
    background-color: rgb(0, 191, 255);
}

/* 3. position: absolute */
.caja-absolute {
    position: absolute;
    top: 10px;
    right: 20px;
    background-color: rgb(255, 102, 0);
}

/* 4. position: sticky */
.caja-sticky {
    position: sticky;
    background-color: rgb(255, 209, 2);
}

/* 5. position: fixed */
.caja-fixed {
    position: fixed;
    background-color: rgb(185, 68, 253);
}
