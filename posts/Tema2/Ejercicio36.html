Ejercicio 36 — Técnicas de centrado y alineación

📅 21 de noviembre de 2025 ✍️ Pablo Jiménez 📚 Tema 2

En este ejercicio se practican distintas técnicas de centrado y alineación utilizando únicamente CSS clásico. Se trabaja el centrado horizontal, centrado de texto, alineación con float y centrado vertical mediante padding y line-height.

Enunciado

  • Centrar un elemento de bloque horizontalmente.
  • Centrar texto dentro de un contenedor.
  • Centrar una imagen horizontalmente.
  • Alinear elementos a la izquierda y a la derecha usando distintos métodos.
  • Centrar verticalmente un elemento usando padding y line-height.

Centrado de un elemento de bloque

Para centrar un elemento de bloque se utiliza margin: auto. El navegador reparte automáticamente el espacio sobrante a ambos lados. 

.block {
  width: 300px;
  margin: 20px auto;
}

Centrado de texto

El texto se centra usando la propiedad text-align: center, que afecta al contenido inline del contenedor.

Centrado de imágenes

Las imágenes se centran convirtiéndolas en elementos de bloque y aplicando margin: auto. 

.image-container img {
  display: block;
  margin: 0 auto;
}

Alineación izquierda y derecha

En este ejercicio se utilizan elementos flotantes con float: left y float: right para colocar contenido a ambos lados del contenedor.

Centrado vertical

El centrado vertical se consigue combinando padding y line-height. Esta técnica funciona bien cuando el contenido es de una sola línea. 

.vertical-center {
  height: 100px;
  padding: 30px 0;
  line-height: 40px;
}

Preguntas del ejercicio

¿Qué sucede si se quita margin: auto?

El elemento deja de centrarse horizontalmente y se coloca alineado a la izquierda, que es el comportamiento por defecto de los elementos de bloque.

¿Cómo afecta la altura al centrado vertical con line-height?

El valor de line-height debe estar en proporción con la altura del contenedor. Si no coincide, el texto no se verá correctamente centrado y puede quedar visualmente desajustado.

Conclusión

Este ejercicio muestra diferentes formas clásicas de alineación en CSS. Aunque hoy en día se utilizan Flexbox y Grid para estos casos, comprender estas técnicas es fundamental para entender el flujo y el comportamiento de los elementos en CSS.
