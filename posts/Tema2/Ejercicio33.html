Ejercicio 33 — Control del orden con z-index

Publicado el 21 de noviembre de 2025 por pjimpin1207

En este ejercicio se trabaja la propiedad z-index, que permite controlar el orden de apilamiento de los elementos posicionados en CSS. Gracias a esta propiedad, es posible decidir qué elementos se muestran por encima o por debajo de otros cuando se superponen.

Enunciado

  1. Crear una página web con tres cuadrados.
  2. Utilizar la propiedad z-index para controlar el orden de apilamiento.
  3. Modificar los valores de z-index en el CSS para cambiar qué cuadrado aparece encima.

Explicación

Para que z-index funcione correctamente, los elementos deben tener una propiedad position distinta de static. En este caso, los tres cuadrados están posicionados de forma absolute y se superponen parcialmente.

El cuadrado azul tiene el valor de z-index más alto, por lo que aparece encima del resto. El verde se sitúa en un nivel intermedio y el rojo queda en la parte inferior.

Código HTML

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Ejercicio z-index</title>
    <link rel="stylesheet" href="Ejercicio33.css">
</head>
<body>

    <div class="cuadro rojo"></div>
    <div class="cuadro verde"></div>
    <div class="cuadro azul"></div>

</body>
</html>

Código CSS

.cuadro {
    width: 120px;
    height: 120px;
    position: absolute;
}

/* Primer cuadrado */
.rojo {
    background-color: rgb(255, 142, 142);
    top: 50px;
    left: 50px;
    z-index: 1;
}

/* Segundo cuadrado */
.verde {
    background-color: rgb(0, 255, 0);
    top: 90px;
    left: 90px;
    z-index: 2;
}

/* Tercer cuadrado */
.azul {
    background-color: rgb(0, 183, 255);
    top: 130px;
    left: 130px;
    z-index: 3;
}
