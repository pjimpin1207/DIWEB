Ejercicio 17 — Propiedades de Texto en CSS

📅 14 de noviembre de 2025 ✍️ Pablo Jiménez 📚 Tema 2

En esta actividad se practica cómo las propiedades de texto en CSS afectan directamente a la apariencia, legibilidad y jerarquía visual del contenido. Se han aplicado diferentes valores para color, font-size, text-transform, text-align, letter-spacing, text-indent, font-style y background-color para personalizar encabezados, párrafos y bloques de cita.

Enunciado

A partir del código HTML y CSS proporcionado:

  • Aplica colores contrastados como #333333 o #007acc.
  • Usa tamaños en píxeles (ej. 32px, 24px).
  • Alinea texto con center o justify.
  • Aplica transformaciones como uppercase o capitalize.
  • Modifica el espaciado entre letras con letter-spacing.
  • Subraya texto con text-decoration: underline.
  • Indenta párrafos con text-indent: 30px.
  • Aplica italic y un fondo suave con rgba() al blockquote.

Código HTML

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Práctica de Propiedades de Texto en CSS</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <h1>Propiedades de Texto en CSS</h1>
    <h2>Subtítulo Ejemplo</h2>
    <p>Este es un párrafo de ejemplo en el que aplicarás distintas propiedades de texto...</p>
    <blockquote>Este es un bloque de cita...</blockquote>
</body>
</html>

Código CSS

body {
  font-family: Arial, sans-serif;
  line-height: 1.5;
}
h1 {
  color: #333333;
  font-size: 32px;
  text-transform: uppercase;
  text-align: center;
}
h2 {
  color: #555555;
  font-size: 24px;
  letter-spacing: 2px;
  text-decoration: underline;
}
p {
  font-size: 16px;
  color: #666666;
  text-indent: 30px;
  text-align: justify;
}
blockquote {
  font-style: italic;
  color: #444444;
  padding: 10px;
  background-color: #f0f8ff;
  border-left: 3px solid #007acc;
}
