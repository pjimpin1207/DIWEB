Ejercicio 35 — Uso de float para envolver texto

Publicado el 21 de noviembre de 2025 por pjimpin1207

En este ejercicio se trabaja una técnica clásica de maquetación en CSS: el uso de la propiedad float para permitir que el texto fluya alrededor de una imagen, simulando el diseño típico de artículos editoriales.

Enunciado

  • Hacer que la imagen del artículo flote a la derecha usando float: right.
  • Observar cómo los párrafos envuelven la imagen flotante.
  • Aplicar clear: both a la nota final para evitar solapamientos.
  • Comprobar cómo la nota de pie se sitúa debajo de la imagen flotante.

Explicación

Al aplicar float: right a la imagen, esta sale del flujo normal del documento y se coloca a la derecha del contenedor. El texto de los párrafos se adapta automáticamente, envolviendo la imagen.

Sin embargo, los elementos posteriores también intentan colocarse a su lado. Para evitar este comportamiento en la nota de pie, se utiliza clear: both, forzando que dicho elemento empiece debajo de cualquier elemento flotante.

Código HTML

<div class="article-container">
  <h1>El Renacimiento de las Micro-Comunidades Digitales</h1>

  <img class="article-image" src="imagen.jpg" alt="Imagen del artículo">

  <p>
    Párrafo 1: El texto rodea a la imagen flotante gracias al uso de float.
  </p>

  <p>
    Párrafo 2: El contenido continúa adaptándose al espacio disponible.
  </p>

  <div class="footer-note">
    Nota de pie: este contenido debe aparecer debajo de la imagen flotante.
  </div>
</div>

Código CSS

.article-image {
    float: right;
    margin-left: 15px;
    margin-bottom: 10px;
    width: 250px;
}

.footer-note {
    clear: both;
    margin-top: 30px;
}

Conclusión

Este ejercicio demuestra cómo float permite crear diseños editoriales clásicos y cómo clear es fundamental para evitar problemas de solapamiento. Aunque hoy en día se usan más Flexbox y Grid, estas propiedades siguen siendo importantes para comprender el flujo de los elementos en CSS.
