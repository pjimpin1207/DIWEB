Ejercicio 8 — Selectores por clase e ID

Publicado el 05 de noviembre de 2025 por Pablo Jiménez

Este ejercicio tiene como propósito practicar el uso combinado de selectores por clase y por ID en hojas de estilo CSS, diferenciando cómo se aplican los estilos a varios elementos HTML. (Puedes reemplazar este texto con tu propia descripción más detallada.)

Código HTML

<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Ejercicio 8</title>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <h2 id="primero" class="impar">Este encabezado h1 será de color azul</h2>
  <h2 class="par">Este encabezado h1 será de color azul</h2>
  <h2 class="impar">Este encabezado h1 será de color azul</h2>
  <h2 class="par">Este encabezado h1 será de color azul</h2>
</body>
</html>

Código CSS

.par {
    color: red;
}

.impar {
    color: green;
}

#primero {
    color: blue;
}
