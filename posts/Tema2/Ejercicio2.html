Ejercicio 2 — Tipos de estilos CSS

Publicado el 04 de noviembre de 2025 por Pablo Jiménez

En este ejercicio se practica el uso de los tres tipos principales de estilos CSS: externo (en un archivo .css aparte), interno (dentro de la etiqueta <style>), y en línea (usando el atributo style directamente sobre un elemento).

Código HTML

<!doctype html>
<html lang="es">
<head>
  <meta charset="utf-8">
  <title>Ejercicio 2.2</title>

  <!--Estilo externo-->
  <link rel="stylesheet" href="Ejercicio2.css">

  <!--Estilo interno-->
  <style>
    p {
      background-color: lightblue;
    }
  </style>
</head>
<body>
  <!--Estilo en la línea-->
  <p style="background-color: pink;">
    El deporte es una actividad física o mental que contribuye al bienestar integral, ya que no solo mejora la salud física al fortalecer músculos y huesos, prevenir enfermedades y controlar el peso, sino que también aporta beneficios mentales y sociales. Al practicarlo, se desarrollan valores como la disciplina, el respeto y el trabajo en equipo, se combate el estrés y se mejora el estado de ánimo y la autoestima. Desde la recreación hasta la competición, el deporte es esencial para una vida saludable, prolongada y de mayor calidad
  </p>
</body>
</html>

Código CSS

p {
  background-color: lightgreen;
}
