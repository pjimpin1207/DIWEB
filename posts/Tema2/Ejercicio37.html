Ejercicio 37: Propiedad overflow en CSS

CSS · Maquetación · Control del desbordamiento

Enunciado

En este ejercicio se trabaja con la propiedad overflow de CSS para controlar qué ocurre cuando el contenido de un elemento supera el tamaño del contenedor.

  • Crear un contenedor de tamaño fijo con contenido largo.
  • Aplicar diferentes valores a la propiedad overflow.
  • Configurar los ejes overflow-x y overflow-y por separado.

Explicación

La propiedad overflow define cómo se muestra el contenido que se desborda del tamaño asignado a un elemento. Es especialmente útil cuando se trabaja con contenedores de tamaño fijo.

Los valores utilizados en este ejercicio son:

  • hidden: el contenido sobrante se recorta y no es visible.
  • scroll: se muestran barras de desplazamiento.
  • visible: el contenido se muestra fuera del contenedor.

Estructura HTML

El HTML contiene tres contenedores con el mismo contenido pero con distinto comportamiento visual según la propiedad overflow. 


<div class="contenedor1">
    <p>Contenido largo...</p>
</div>

<div class="contenedor2">
    <p>Contenido largo...</p>
</div>

<div class="contenedor3">
    <p>Contenido largo...</p>
</div>

Estilos CSS

En el archivo CSS se define el tamaño fijo del contenedor y se prueban diferentes valores de overflow, así como el control independiente del eje horizontal y vertical. 


.contenedor1 {
    width: 400px;
    height: 200px;
    overflow: hidden;
    overflow-x: hidden;
}

.contenedor2 {
    width: 400px;
    height: 200px;
    overflow: scroll;
    overflow-y: scroll;
}

.contenedor3 {
    width: 400px;
    height: 200px;
    overflow: visible;
}

Resultado

Al visualizar la página:

  • El primer contenedor recorta el contenido sobrante.
  • El segundo muestra barras de desplazamiento.
  • El tercero permite que el contenido sobresalga.

Este ejercicio permite comprender cómo controlar el desbordamiento del contenido y cuándo es conveniente permitir o limitar el scroll dentro de un elemento.
