Proyecto La 10 Store

Publicado el 31 de octubre de 2025 por Pablo Jiménez

Este proyecto se diseña una tienda online de camisetas de fútbol llamada La 10 Store, abordando la planificación, estructura y diseño de interfaces adaptadas a usuarios clientes y administradores.

1. Mapa conceptual

Se desarrolló un mapa conceptual que describe las secciones principales de la tienda: Inicio, Catálogo, Carrito, Proceso de compra, Cuenta de usuario y Administración. Este mapa sirve de base para la navegación general.

2. Funcionalidades de usuarios

  • Clientes:
    • Navegación y búsqueda (por equipo, jugador, temporada, precio...)
    • Visualización de producto (imágenes, tallas, descripción, disponibilidad)
    • Carrito de compras (añadir, modificar o eliminar camisetas)
    • Proceso de compra (registro, dirección, método de pago, confirmación)
    • Gestión de cuenta (perfil, historial, seguimiento, contraseña)
    • Atención al cliente (FAQ, contacto y devoluciones)
  • Administradores:
    • Gestión de productos (añadir, editar, eliminar, gestionar stock)
    • Gestión de pedidos (procesar, enviar, entregar, reembolsar)
    • Gestión de usuarios (ver clientes y permisos)

3. Estructura de navegación

Se diseñó un mapa jerárquico que refleja la relación entre las secciones: Inicio → Catálogo → Detalle de producto → Carrito → Compra → Cuenta de usuario.
También se incluye una zona de Administración exclusiva para vendedores.

4. Recursos de diseño

El diseño se realizó utilizando Figma, con diferentes fases:

  • Wireframe
  • Mockup
  • Prototipo
  • Guía de estilo

5. Mejoras implementadas

  • Mapa de navegación individual para clientes y administradores.
  • Diseño responsive adaptado a móvil, tablet y escritorio.
  • Versión navegable en HTML del mapa de navegación.
  • Optimización de imágenes para una carga rápida.
  • Uso de menú hamburguesa en dispositivos móviles.
